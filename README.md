## ✨ About

QuickAttend Landing is the official product showcase website for **QuickAttend**.

The website highlights:

* AI-powered attendance workflow
* Teacher journey
* Student journey
* Core features
* Technology stack
* Direct access to the live application

Designed with a modern and responsive interface focused on simplicity and user experience.

## ▶️ Run Locally

Clone repository:

```bash
git clone https://github.com/lakshaybamel/quick-attend-landing.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run:

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
